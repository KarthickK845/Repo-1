primitive and non-primitive diff
null value can be set for non-primitive datatypes but not for primitive

static variables are allocated memory during - compile time; JVM is responsible for setting default value to static variables.
instance variables are allocated memory during runtime; 
A obj = new A();
obj is called reference variable.
object is created during 'new A();'
